OpenProject Analytics

This project analyzes milestone structure, effort estimation, and risk exposure based on OpenProject exports. The data pipeline is implemented in Python, while the visualization layer uses D3.js.

Project Context

The goal is to transform project management data into actionable insights. Milestones, story points, and risks are combined into quantitative metrics to support prioritization and decision-making.

Milestone Overview

Milestone–Risk Relationships

Methodology

  • Data extraction from OpenProject (XLS export)
  • Parsing and normalization in Python
  • Risk weighting based on probability
  • Computation of combined effort-risk scores
